默认快捷键

若没有小键盘，可通过"File"->"Settings"->"Keymap"->"Plugins"->"W-Reader"前往修改快捷键。

功能 快捷键
搜索 Alt + NumPad-7(小键盘7)
目录 Alt + NumPad-9(小键盘9)
加载本地文件 Alt + NumPad add(小键盘+号)
编辑器提示消息 Alt + NumPad-0(小键盘0)
代码注释块 Alt + NumPad-1(小键盘1)
上一章 Alt + NumPad-4(小键盘4)
下一章 Alt + NumPad-6(小键盘6)
上一行 Alt + NumPad-8(小键盘8)
下一行 Alt + NumPad-2(小键盘2)
常见问题
  • 1.侧边栏不小心关闭，不显示内容：
    通过"File"->"Settings"->"Tools"->"W-Reader"进入设置页面， 将类型切换至其它（点击Apply按钮）后再切换回"侧边栏"即可。
  • 2."编辑器提示消息"功能不起作用：
    代码编辑器中必须有焦点，并且编辑器中要有文本插入符（闪烁的竖线）存在，才能使用此功能。
    ps: 多个文本插入符存在时，最后设置的文本插入符为主文本插入符，只有主文本插入符才会起作用。
  • 3."代码注释块"功能不起作用：
    与问题2类似，但有一个额外的条件，就是当前编辑器所选择的文件处于可写状态。
    ps: 多个文本插入符存在时，最后设置的文本插入符为主文本插入符，只有主文本插入符才会起作用。
