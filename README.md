### 这是一个JetBrains IDEs上的小说阅读器插件。
插件地址：https://plugins.jetbrains.com/plugin/25572-w-reader
#### 核心功能
###### 便捷检索，快速定位： 支持在线搜索海量小说资源，精准定位心仪作品。
###### 多元书源，随心切换： 整合多个优质书源，覆盖热门网文与经典名著。
###### 本地加载，兼容并蓄： 无缝支持 TXT/EPUB 格式本地小说加载。
###### 模式切换，自由随心： 支持侧边栏/状态栏模式切换，适配不同屏幕布局。
###### 独特阅读，专属体验： 支持编辑器提示消息、代码注释块模式阅读。
###### 自动阅读，解放双手： 开启自动阅读功能，内容自动切换播放。
###### 听书功能，声色俱全： 支持多音色听书模式，享受听觉盛宴。
###### 快捷键控，高效阅读： 预设翻页/章节切换/听书等快捷键，提升阅读效率。
